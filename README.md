# SafeAnchor-Grasp Project Page

Static project page for:

**SafeAnchor-Grasp: Same-Object Safe Enhancement for Dexterous Grasping in Cluttered Multi-Object Scenes**

## Files

- `index.html`: project page content.
- `styles.css`: page layout and responsive styling.
- `assets/manuscript.tex`: manuscript source copied from `real_artical16.tex`.
- `assets/images/`: project figures and hero image.
- `assets/videos/`: real grasp execution videos.

Open `index.html` directly in a browser, or run a local static server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.
